# Conversion-Tools-GUI
GUI for conversion of PDF to PNG/Word, and text extraction from Img to Word

## How to Use
### Download
```
git clone https://github.com/lottielin/Conversion-Tools-GUI.git
```
### Run the program
```
python convTools.py
```
